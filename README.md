# CANDY_AVB
## Description
Intel MAX10(10M16SA) board for 100Base-T Ethernet and DSP.

## Specifications
* FPGA: Intel 10M16SAU(MAX10) x1
* ETH: TI DP83848 x1
* DSP: Analog Devices ADAU1761 x2
* USB Serial: Silabs CP2102N x1
* CLOCK: 48MHz Crystal
